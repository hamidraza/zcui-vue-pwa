#### Installation

``` bash
$ npm install --global zcui
$ zcui init HelloWorld --target vue-pwa
```

``` bash
$ cd HelloWorld
```

#### Available commands

``` bash
# start development server
$ npm run dev

# create production build
$ npm run prod

#analyze dependencies
$ npm run analyze
```
